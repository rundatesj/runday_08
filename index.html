🚀 30초만에 완벽한 시스템 구축하기

1️⃣ Google Forms 생성

  • forms.google.com 접속
  • 새 양식 만들기
  • 질문 추가: 닉네임, 날짜, 거리
  • 응답을 스프레드시트로 연결

2️⃣ 스프레드시트 공개

  • 연결된 시트에서 "파일 > 웹에 게시"
  • 형식: CSV 선택
  • URL 복사
  • 아래 입력란에 붙여넣기

3️⃣ Forms 임베드

  • Forms에서 "보내기" 클릭
  • < > 아이콘 클릭
  • 임베드 코드 복사
  • 아래 입력란에 붙여넣기
